# Aeromate

> An overambitious student's open-source reimagining of Airbus UpNext's Auto'Mate 
> program — autonomous air-to-air rendezvous, in software, on a student budget. 
> History remembers the ambitious. Airbus, please don't sue me.

## What is this?
Inspired by the [Airbus UpNext](https://www.airbus.com/en/innovation/innovation-ecosystem/airbus-upnext) Auto'Mate program, 
aeromate is a student-led software simulation of autonomous air-to-air rendezvous. 
This is an independent project with no affiliation with or endorsement by Airbus.
